Face Recogntion App

Built a full-stack face recognition application utilizing React for the front end. Established a robust server using NodeJS with meticulously implemented routing, complemented by a fully functional PostgreSQL database. The database securely stores user information, including hashed passwords, achieved through the proper implementation of the Bcrypt library. The application leverages the Clarifai API for face detection. Users can register, sign in, and submit images for analysis, and keep track of their entries with the results displayed on the frontend. The React components ensure a seamless user experience, and the server is hosted on Heroku.

Canadian Housing Price Predictor

Developed a Canadian housing price forecasting tool utilizing the Java Swing library. Successfully retrieved data from Statistics Canada on New Housing Price Index values, and stored it in a MySQL database. Utilized SQL queries in the Java environment to effectively manipulate and analyze data. The application allows users to select different provinces and analyze the NHPI from the selected date range with various graph analytics displayed using the JFreeChart library. Employed the Weka machine learning library, leveraging a diverse range of machine learning algorithms including Simple Linear Regression and Multi-Layer Perceptron to accurately forecast the future housing price for the user selected region and time.
